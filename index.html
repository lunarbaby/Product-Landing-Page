Features

Elementary OS is a fast operating system. Apps load quickly and remember where you left off. Plus, elementary OS doesn't slow down with updates.

Elementary OS is an open source operating system. All of elementary is available for review, scrutiny, modification, and redistribution by anyone.

Elementary OS is build on GNU/Linux, which is one of the most secure systems in the world.
